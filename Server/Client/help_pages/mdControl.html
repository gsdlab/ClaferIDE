Instance Generator

In this window, you can select, run, and control an instance generator.
  • Use the "" drop down to select either the Alloy-based or Choco-based instance generator.
  • Use the button "" to begin a session with the selected instance generator.
  • Use the button "" to kill a session with a running instance generator.
  • Use the button "" to ask for a next instance.
  • Use the button "" to reload the model from the editor to the currently running instance generator session.
  • Use the button "" to finish the session.
  • Use the button "" to set the global scope or the scope for the clafer with the given name.
  • Use the button "" to increase the global scope or the scope for the clafer with the given name.
