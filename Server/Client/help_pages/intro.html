In this tool, you can edit, compile, explore, and instantiate Clafer models. Press the '?' button of each window to see specific help. In general:
  • Use the view Input Clafer Model and Options to edit and compile a model.
  • Use the view Compiled Formats to explore a HTML rendering of a model.
  • Use the view Instance Generator to run an instance generator, adjust the global and the individual scopes, and the bitwidth.
  • Use the view Output to see the generated instances, messages, and UnSAT cores.
