From this window you can control the Clafer instance generator (ClaferIG).
  • Use the button "" to generate another instance by ClaferIG.
  • Use the button "" to increase the global scope by the provided number.
    * ClaferIG limits the number of instances of each clafer to the provided scope.
    * Note that you cannot reduce scope.
  • The current scope is also displayed.
