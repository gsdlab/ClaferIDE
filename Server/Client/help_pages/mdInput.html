Input Clafer Model and Options

From this window, you can upload your .cfr file, select a predefined example, or enter the model directly using the source code editor.
  • The model must be expressed using Clafer.
  • Use the file selector to choose a file. Make sure the extension of the file you are trying to submit is .cfr. Use the button "" next to the file selector to upload and compile the selected file.
  • Use the "" drop down to select an example and use the button "" next to the example selector to compile the example. Check the checkbox "Load into editor" to have the example loaded into the source code editor. Otherwise, the example will be compiled without affecting the contents of the editor. This is useful for using the examples as a reference.
  • Use the button "" directly above the source code editor to compile and instantiate the model from the editor.
  • Use the "" drop down to select a scope computation strategy: None, Fast (default), and Full.
