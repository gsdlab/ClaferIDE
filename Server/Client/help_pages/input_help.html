Input

From this window, you can choose a .cfr file to optimize.
  • The model must be in an attributed feature model with inheritance subset of Clafer and it must contain optimization goals.
  • Make sure the extension to the file you are trying to submit is ".cfr".
  • Use the button .
  • From this window, you can also choose a .data file with configurations of variants to add to the current visualization.
  • The .data file can be obtained using the button in Variant Comparer view or by running ClaferIG with parameters "--useuids --addtypes" and removing all ": clafer", ": integer", and ": string" supertypes.
